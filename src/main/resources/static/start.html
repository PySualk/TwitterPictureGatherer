Getting Started

  1. As a first step please open the file application.properties and modify your configuration here.
  2. Configure your MongoDB connection by entering hostname and port and optionally username and password.
  3. Configure the local directory where the pictures are being stored (the application only gatheres tweets with pictures).
  4. Enter your consumer key, consumer secret, access token and access secret obtained from Twitter. If you did not have them yet register with your Twitter account at https://dev.twitter.com/.
  5. Now you are able to build a standalone jar file with your configuration by running mvn clean install or simply run the application with mvn spring-boot:run
  6. Define some jobs, so the application is able to gather some data from Twitter for you.
  7. By pressing Start Gathering Tweets the application starts collecting Tweets from the Twitter Streaming API. All Tweets that match your defined jobs are being stored at the MongoDB database. The corresponding pictures are being stored at the local directory you defined earlier.
